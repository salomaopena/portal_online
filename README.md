## [Tecnologias e Bibliotecas](#tecnologias-e-bibliotecas)  

* express: framework web minimalista e flexível para Node.js. Criar servidores, rotas, middlewares e APIs com facilidade.
* express-session: middleware que permite criar e gerenciar sessões de usuários (login, carrinhos de compra etc.). Autenticação e persistência de dados entre requisições.
* express-handlebars: motor de templates para gerar HTML dinâmico usando Handlebars com Express. Renderizar páginas com dados vindos do backend.
* cookie-parser: middleware para ler e manipular cookies enviados pelo navegador. Trabalhar com autenticação, preferências do usuário, sessões etc.
* cors: middleware que habilita CORS (Cross-Origin Resource Sharing). Permitir que frontends hospedados em outros domínios façam requisições para seu servidor.
* path: módulo nativo do Node.js para lidar com caminhos de arquivos e diretórios. Juntar, resolver ou normalizar caminhos entre sistemas operacionais.
* dotenv: Carrega variáveis de ambiente de um arquivo .env para o process.env. Armazenar configurações sensíveis (senhas, tokens, URLs) fora do código.
* jsonwebtoken: biblioteca para gerar e verificar tokens JWT (JSON Web Token). Autenticação baseada em token (login seguro, APIs protegidas).
* bcryptjs: biblioteca para criptografar senhas com hash seguro. Armazenar senhas de forma segura em bancos de dados.
* nodemailer: módulo para envio de emails diretamente do servidor Node.js. Envio de notificações, confirmações, recuperação de senha, etc.
* mysql2: cliente MySQL moderno e mais rápido para Node.js. Conectar e fazer consultas ao banco de dados MySQL/MariaDB.
* nodemon: ferramenta que reinicia automaticamente o servidor Node.js quando há alterações no código. Facilitar o desenvolvimento sem precisar reiniciar manualmente.
* multer: middleware para lidar com o upload de arquivos em formulários multipart/form-data. Upload de imagens, documentos e outros arquivos para o servidor.
